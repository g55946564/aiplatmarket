# 08. LifeMap 프로젝트와의 연동

> AI인생·진로(LifeMap, module id: `life`)는 AI플랫마켓의 19개 학문 분야 기반 분석 플랫폼입니다.
> 이 문서는 LifeMap이 전체 시스템(AI Core, ItemZone, Community)과 어떻게 맞물리는지 정리합니다.

---

## 8.1 LifeMap이란

개인의 생년월일·관심사·고민 등을 입력받아, 19개 학문 분야(사주/별자리/MBTI/심리학/적성/시장성 등)
관점에서 AI가 종합 분석을 제공하는 모듈입니다. `module.json`의 id는 `life`, group은 `platform`입니다.

## 8.2 AI Core 연동 방식 (이미 구현됨)

```js
// 학문 분야별로 engineMap.js의 lifeai 섹션에 우선순위가 등록되어 있음
const res = await AICore.process({
  module: 'lifeai',
  task: 'mbti',          // 19개 분야 중 하나
  prompt: '...',
  fallbackFn: async () => '규칙 기반 분석 결과',
});
```

- 사주/별자리/풍수 등 **전통 명리 계열은 Rule(규칙 기반) 우선** — 정해진 계산 로직이 더
  일관적이기 때문입니다.
- 적성/심리/시장성 등 **현대적 분석 계열은 LLM(GPT/Claude/Gemini) 우선**입니다.

## 8.3 ItemZone과의 연계 지점

LifeMap의 "적성 분석"과 ItemZone의 "사업성 분석"은 같은 AI Core를 쓰지만, 다음과 같이
**연계 가능한 시나리오**가 설계되어 있습니다 (일부는 향후 구현):

- LifeMap에서 "창업 적성이 높다"는 분석 결과가 나온 사용자에게, ItemZone "아이템 등록"으로
  자연스럽게 유도하는 CTA(Call To Action) 배치
- ItemZone에서 사업을 등록한 사용자가, 자신의 LifeMap 적성 분석 결과를 사업계획서 보조 자료로
  함께 첨부할 수 있는 기능 (설계 단계)

## 8.4 Community와의 연계 지점

- LifeMap 분석 결과 공유 게시판 (커뮤니티 cm-tab에 향후 "라이프맵 후기" 카테고리 추가 가능 —
  현재 카테고리: 자유/정보/Q&A/공구/후기/상생회/모임)
- "비슷한 적성을 가진 사람들" 매칭 → 모임 만들기 기능과 연계 (8.5 참조)

## 8.5 모임 만들기 기능과의 연계 (향후)

커뮤니티에 추가된 "🙋 모임 만들기" 기능은 향후 LifeMap의 적성/관심사 분석 결과를 활용해
"비슷한 사업 관심사를 가진 사람들" 모임을 추천하는 기능으로 확장할 수 있습니다 (설계 단계,
개인정보 활용 동의 절차 필요).

## 8.6 데이터 흐름 요약

```
사용자 입력 (생년월일·관심사·고민)
        ↓
AICore.process({module:'lifeai', task:'...'})
        ↓
19개 학문 분야 결과 종합
        ↓
┌───────────────┬──────────────────┐
│ Fusion Score   │  적성→ItemZone 유도  │
│ (종합 점수 산출)  │  CTA / 모임 추천(향후) │
└───────────────┴──────────────────┘
```
