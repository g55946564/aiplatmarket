# AI_BOOTSTRAP.md

> 이 파일은 60초 안에 "내가 지금 뭘 하는 프로젝트에 합류했는지"를 이해시키기 위한 진입점입니다.
> 더 깊은 내용은 이 문서가 안내하는 다음 파일들을 순서대로 읽으면 됩니다.

## 당신은 지금 AI플랫마켓(AIplatMarket) 개발에 참여했습니다

한 줄 정의: **소상공인과 소비자를 AI로 연결하는, 100개 이상의 독립 모듈을 수용하는 통합 비즈니스 플랫폼.**

이 프로젝트는 **사람 중심 철학**을 가지고 있습니다. AI는 사람을 대체하는 것이 아니라
사람의 참여와 협업을 돕는 시스템입니다. 자세한 철학은 `SYSTEM_PHILOSOPHY.md`에 있습니다.

## 다음에 읽을 순서 (필요한 만큼만)

| 순서 | 파일 | 언제 읽나 |
|---|---|---|
| 1 | `MASTER_CONTEXT.md` | 전체 구조를 알아야 할 때 (항상 추천) |
| 2 | `MODULE_DEV_GUIDE.md` | 독립 모듈(AIR&D, LifeMap 등)을 직접 개발할 때 |
| 3 | `SYSTEM_PHILOSOPHY.md` | 기능보다 "왜 이렇게 설계했는지"가 궁금할 때 |
| 4 | `PROJECT_STATUS.json` | 지금 뭐가 완성됐고 뭐가 남았는지 확인할 때 |
| 5 | `../../docs/` 전체 | 위 4개로 부족한 깊은 내용이 필요할 때 (배포, 시뮬레이션, AGI 로드맵 등) |

## 지금 당장 알아야 할 3가지 규칙

1. **모든 AI 호출은 `ai-core/`(ChatDoumi AI Core) 를 통해서만 한다.** GPT/Claude/Gemini를 직접 호출하지 않는다.
2. **framework/ 폴더는 수정하지 않는다.** 모듈 개발자는 framework를 "사용"만 한다.
3. **미완성 기능은 오류 대신 "업그레이드 중" 화면을 보여준다.**

## 실시간 상태가 필요하면

```
GET https://aiplatmarket.com/api/ai-context
```
이 문서들(정적 markdown)은 "구조와 철학"을 담고, 위 API는 "지금 이 순간의 상태"를 담습니다.
둘을 합치면 새 대화창도 메인 개발창과 동일한 그림을 봅니다.
