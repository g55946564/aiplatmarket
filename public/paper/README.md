# 국민경제㊉ (life-Info)

본 저장소는 **국민경제㊉**의 디지털 신문 뷰어 시스템입니다. 
지역 생활 정보와 복지 소식을 독자들에게 '종이 신문 그대로'의 경험으로 전달합니다.

## 📰 주요 기능
- **신문 그대로 보기:** 타블로이드 판형 최적화 뷰어
- **스마트 확대/축소:** 모바일 환경에서의 가독성 극대화 (Pinch Zoom & Double Tap)
- **아카이브 서비스:** 지난 호 다시보기 제공
- **챗도우미 연동:** 지면 읽기 중 실시간 상담 연결

## 📂 파일 관리 규칙
- `index.html`: 신문 전용 뷰어 메인 엔진
- `life-Info.pdf`: 최신호 신문 데이터 (매주 업데이트)
- `thumb.jpg`: 메인 페이지 노출용 최신호 표지 이미지
- `/archive`: 지난 신문 보관소
